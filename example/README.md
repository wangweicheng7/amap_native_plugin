# amap_native_plugin example

Demonstrates common `amap_native_plugin` usage flows.

The home page lists examples and opens each one as a second-level page:

- Basic map display.
- Fixed-size map widget.
- Static circle geofence drawing.
- Map-center circle geofence editing with a radius slider.

Run with your AMap API keys:

```sh
flutter run \
  --dart-define=AMAP_ANDROID_API_KEY=your-android-key \
  --dart-define=AMAP_IOS_API_KEY=your-ios-key
```
